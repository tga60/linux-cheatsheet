Linux Command Cheat Sheet

A quick reference guide for common Linux commands

File System Commands

ls
List directory contents
ls -la
cd
Change directory
cd /path/to/directory
pwd
Print working directory
pwd
mkdir
Create new directory
mkdir new_directory
rm
Remove files or directories
rm -rf directory
cp
Copy files or directories
cp file.txt /destination/
mv
Move or rename files/directories
mv file.txt new_name.txt
touch
Create empty file or update timestamp
touch newfile.txt
find
Search for files in directory hierarchy
find /home -name "*.txt"
cat
Print file contents to standard output
cat file.txt
less
View file contents with pagination
less large_file.log
head
Display first lines of a file
head -n 10 file.txt
tail
Display last lines of a file
tail -f /var/log/syslog
nano
Simple text editor
nano file.txt
ln
Create links between files
ln -s target_file link_name

Process Management Commands

ps
Report a snapshot of current processes
ps aux
top
Display dynamic real-time view of running processes
top
htop
Interactive process viewer (enhanced version of top)
htop
kill
Terminate processes by ID
kill -9 1234
killall
Kill processes by name
killall firefox
pkill
Kill processes by name or other attributes
pkill -f "process_pattern"
bg
Send process to background
bg %1
fg
Bring process to foreground
fg %1
jobs
List active jobs
jobs -l
nohup
Run command immune to hangups
nohup command &
nice
Run a program with modified scheduling priority
nice -n 10 command
renice
Alter priority of running processes
renice -n 10 -p 1234

Networking Commands

ifconfig
Configure network interface parameters
ifconfig eth0
ip
Show/manipulate routing, devices, policy routing and tunnels
ip addr show
ping
Send ICMP ECHO_REQUEST to network hosts
ping google.com
netstat
Print network connections, routing tables, interface stats
netstat -tuln
ss
Utility to investigate sockets (replacement for netstat)
ss -tuln
traceroute
Print the route packets trace to network host
traceroute google.com
nslookup
Query Internet name servers
nslookup google.com
dig
DNS lookup utility
dig google.com
wget
Non-interactive network downloader
wget https://example.com/file
curl
Transfer data from or to a server
curl -O https://example.com/file
ssh
OpenSSH SSH client (remote login program)
ssh user@hostname
scp
Secure copy (remote file copy program)
scp file.txt user@host:/path
rsync
Fast, versatile file copying tool
rsync -av source/ dest/
iptables
Administration tool for IPv4 packet filtering and NAT
iptables -L
nc
Netcat - networking utility for reading/writing network connections
nc -l 1234

Permissions & Ownership Commands

chmod
Change file mode bits (permissions)
chmod 755 file.txt
chown
Change file owner and group
chown user:group file.txt
chgrp
Change group ownership
chgrp group file.txt
umask
Set file mode creation mask
umask 022
sudo
Execute a command as another user, typically superuser
sudo apt update
su
Change user ID or become superuser
su - username
visudo
Edit sudoers file safely
visudo
getfacl
Get file access control lists
getfacl file.txt
setfacl
Set file access control lists
setfacl -m u:user:rwx file.txt

Package Management Commands

apt
Advanced Package Tool (Debian/Ubuntu)
apt install package_name
apt-get
APT package handling utility (older version)
apt-get update
dpkg
Debian package manager
dpkg -i package.deb
yum
Yellowdog Updater Modified (RHEL/CentOS)
yum install package_name
dnf
Dandified YUM (Fedora/RHEL 8+)
dnf install package_name
rpm
RPM Package Manager
rpm -i package.rpm
pacman
Arch Linux package manager
pacman -S package_name
snap
Universal Linux package manager
snap install package
flatpak
Software deployment and package management for Linux
flatpak install app_name

User Management Commands

useradd
Create a new user
useradd -m username
userdel
Delete a user account
userdel -r username
usermod
Modify a user account
usermod -aG group username
passwd
Change user password
passwd username
groupadd
Create a new group
groupadd group_name
groupdel
Delete a group
groupdel group_name
id
Print user and group information
id username
who
Show who is logged in
who
w
Show who is logged in and what they are doing
w
last
Show listing of last logged in users
last

System Info & Monitoring Commands

uname
Print system information
uname -a
uptime
Tell how long the system has been running
uptime
free
Display amount of free and used memory
free -h
vmstat
Report virtual memory statistics
vmstat 1 5
iostat
Report CPU and I/O statistics
iostat -x 2
lscpu
Display CPU architecture information
lscpu
lsusb
List USB devices
lsusb
lspci
List PCI devices
lspci
lspci
List PCI devices
lspci
dmesg
Print or control the kernel ring buffer
dmesg | grep error
journalctl
Query systemd's journal
journalctl -u service-name
hostname
Show or set system hostname
hostname

Compression & Archiving Commands

tar
Tape archiver utility
tar -czvf archive.tar.gz directory/
gzip
Compress or expand files
gzip file.txt
gunzip
Uncompress gzipped files
gunzip file.txt.gz
bzip2
Block-sorting file compressor
bzip2 file.txt
bunzip2
Uncompress bzip2 files
bunzip2 file.txt.bz2
xz
XZ compression utility
xz file.txt
unxz
Uncompress xz files
unxz file.txt.xz
zip
Package and compress files
zip -r archive.zip directory/
unzip
Extract compressed files in zip format
unzip archive.zip

Disk & Partitioning Commands

df
Report file system disk space usage
df -h
du
Estimate file space usage
du -sh directory/
fdisk
Manipulate disk partition table
fdisk -l /dev/sda
parted
Disk partition manipulator
parted /dev/sda print
mkfs
Build a Linux filesystem
mkfs.ext4 /dev/sda1
fsck
Check and repair a Linux filesystem
fsck /dev/sda1
mount
Mount a filesystem
mount /dev/sda1 /mnt
umount
Unmount filesystems
umount /mnt
lsblk
List block devices
lsblk

Miscellaneous Utilities

grep
Search for patterns in files
grep "pattern" file.txt
sed
Stream editor for filtering and transforming text
sed 's/old/new/g' file.txt
awk
Pattern scanning and text processing language
awk '{print $1}' file.txt
date
Display or change the date & time
date +"%Y-%m-%d %H:%M:%S"
cal
Display a calendar
cal -3
history
Command history
history | grep command
alias
Create command aliases
alias ll='ls -la'
cron
Schedule periodic background work
crontab -e
xargs
Build and execute command lines from input
find . -name "*.log" | xargs rm
watch
Execute a program periodically, showing output
watch -n 1 'free -m'
